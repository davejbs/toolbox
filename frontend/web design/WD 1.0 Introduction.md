# Le Web Design

## Contexte 
On dit souvent qu'il ne faut pas juger un livre par sa couverture, sauf que ça ne s'applique pas quand il s'agit d'un site web. Les utilisateurs mettent très peu de temps avant de se faire une idée sur la compagnie, la marque ou les produits et cela rien qu'à partir du site.

## Le design peut influencer le comportement de la clientèle
Un site bien monté, visuellement cohérent peut plus facilement mettre en confiance la clientèle afin que cette dernière puisse effectuer des actions tels que des achats.

## Les 4 principaux principes du web design
Il existe 4 principaux principes en web design, on y retrouve:

- Colour Theory
- Typography
- User Interface Design
- User Experience Design
